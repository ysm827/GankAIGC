# GankAIGC 启动命令

## 1. 启动数据库

```bash
cd /home/mumu/projects/gankaigc
docker compose --env-file .env.docker -f docker-compose.yml -f docker-compose.local.yml up -d postgres
```

## 2. 启动项目

```bash
cd /home/mumu/projects/gankaigc/package
source venv/bin/activate
python main.py
```

## 3. 打开页面

`python main.py` 启动成功后，会在终端打印服务地址，浏览器打开它即可，例如：

```text
http://172.30.130.23:9800
```

> 不要关闭运行 `python main.py` 的终端；停止服务按 `Ctrl+C`。
